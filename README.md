# assets-jellyfin

Este repositório contém apenas assets para uso com o Jellyfin — imagens, logos, capas, ícones e outros arquivos estáticos relacionados à interface/galeria. Não é um plugin, não contém código do servidor nem configurações do Jellyfin; é apenas um repositório de recursos visuais.

## O que tem aqui
- Logos e ícones
- Capas e banners
- Thumbnails e imagens de exemplo
- Outros arquivos de mídia estática úteis para personalização
